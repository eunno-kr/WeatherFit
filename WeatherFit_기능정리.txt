========================================
  WeatherFit — 구현 기능 전체 정리
  날씨 기반 AI 코디 추천 웹앱
========================================

■ 프로젝트 개요
  - 이름     : WeatherFit
  - 기술 스택 : React 19 + Vite 7 + Tailwind CSS 3
  - 날씨 API  : Open-Meteo (무료, 실시간)
  - AI API    : Groq (Llama 3.3 70B, 무료)
  - 저장 방식 : 브라우저 LocalStorage (서버 없음)
  - 실행 포트 : http://localhost:5175


========================================
  구현 기능 목록
========================================

────────────────────────────────────────
1. 실시간 날씨 기반 코디 추천
────────────────────────────────────────
  - Open-Meteo API로 현재 날씨 실시간 불러오기
  - 기온, 체감온도, 습도, 날씨 상태 표시
  - 최저/최고/일교차 정보 제공
  - 체감온도 기준으로 옷 두께 자동 계산
  - 비/눈 여부에 따라 방수 아이템 추천
  - 일교차 8° 이상 시 레이어링 경고 표시

────────────────────────────────────────
2. 7일 날씨 예보 + 미리 보기
────────────────────────────────────────
  - 오늘 ~ 7일치 날씨 예보 카드 표시
  - 요일 클릭 시 해당 날 날씨 기준으로 코디 전환
  - 내일 강수 확률 50% 이상이면 상단 알림 배너 출력
  - 모바일에서 가로 스크롤 지원

────────────────────────────────────────
3. 3가지 코디 조합 자동 추천 (TODAY'S LOOK)
────────────────────────────────────────
  - 날씨 + 스타일 + 상황 조합으로 3가지 코디 제안
  - 각 코디: 레이어(아우터) / 상의 / 하의 / 슈즈 / 포인트(악세서리)
  - 코디별 추천 이유 태그 표시
  - 코디 저장 버튼 (브라우저에 보관)
  - 코디 공유 버튼 (Web Share API / 클립보드 복사)

────────────────────────────────────────
4. 코디 이미지 카드 저장 (PNG 다운로드)
────────────────────────────────────────
  - 각 코디 카드에 "↓ 카드" 버튼
  - 클릭 시 PNG 이미지 자동 다운로드
  - 이미지 구성: WeatherFit 브랜딩 / 날짜·도시·기온 /
    색상 팔레트 / 코디 아이템 목록 / 푸터
  - SNS 공유, 카카오톡 전송에 바로 사용 가능
  - Canvas API로 브라우저 내에서 직접 생성 (서버 불필요)

────────────────────────────────────────
5. 상황별 추천 (OccasionPicker)
────────────────────────────────────────
  - 5가지 상황 선택: 자유 / 출근 / 데이트 / 운동 / 여행
  - 상황에 따라 추천 스타일 자동 전환
    예) 출근 → 오피스 스타일 / 데이트 → 로맨틱 스타일
  - 선택한 상황이 코디 조합 로직에 반영됨

────────────────────────────────────────
6. 스타일 무드 선택 (MoodPicker)
────────────────────────────────────────
  - 미니멀 / 스트릿 / 캐주얼 / 오피스 / 아웃도어 선택
  - 무드에 따라 색상 조합 및 아이템 스타일 변경

────────────────────────────────────────
7. 내 옷장 등록 및 관리 (WardrobePanel)
────────────────────────────────────────
  - 옷 직접 등록: 이름 / 카테고리 / 색상 / 두께 / 스타일 / 방수 여부
  - 등록한 옷 기반으로 맞춤 코디 추천
  - 옷별 착용 횟수 자동 기록
  - 아이템 고정 기능 (내가 반드시 입을 옷 선택 후 나머지 조합)
  - 계절 정리 알림:
    · 겨울인데 여름옷 비중 35% 초과 → 알림
    · 여름인데 두꺼운 옷 비중 35% 초과 → 알림

────────────────────────────────────────
8. AI 코디 조언 (Groq Llama 3.3 70B)
────────────────────────────────────────
  - "✦ AI에게 물어보기" 버튼 클릭 시 AI 분석
  - AI가 제공하는 내용:
    · 오늘 날씨 주의점
    · 추천 코디 중 가장 적합한 것과 이유
    · 실용적인 스타일링 팁
  - 날씨/상황/옷장 정보를 AI에 전달해 맞춤 답변 생성
  - 날씨·상황이 바뀌면 AI 답변 자동 초기화

────────────────────────────────────────
9. AI 스타일리스트 채팅봇
────────────────────────────────────────
  - 화면 우측 하단 ✦ 플로팅 버튼으로 접근
  - 코디/패션/날씨 관련 자유 질문 가능
  - 빠른 질문 버튼 제공:
    · "오늘 코디 중 뭐가 제일 나아?"
    · "우산 챙겨야 할까?"
    · "오늘 레이어링 어떻게 해?"
    · "포인트 컬러 추천해줘"
  - 대화 히스토리 유지 (멀티턴 대화)
  - 현재 날씨·코디·옷장 정보를 AI가 알고 답변

────────────────────────────────────────
10. 코디 히스토리 기록 (HistoryPanel)
────────────────────────────────────────
  - "오늘 입었어요 ✓" 버튼 클릭 시 기록
  - 날짜 / 코디명 / 스타일 / 상황 / 도시 / 기온 / 날씨 저장
  - 최대 60일치 기록 보관
  - 기록 삭제 기능

────────────────────────────────────────
11. 코디 즐겨찾기 저장 (SavedOutfitsPanel)
────────────────────────────────────────
  - 마음에 드는 코디 저장 버튼
  - 저장된 코디 목록 확인 (최대 30개)
  - 저장 날짜 표시
  - 저장 해제 기능

────────────────────────────────────────
12. 프로필 설정 (ProfileSetup)
────────────────────────────────────────
  - 최초 실행 시 프로필 설정 화면
  - 설정 항목: 나이대 / 성별 / 도시 / 선호 스타일 /
    온도 민감도 / 색감 취향 / 주 색상
  - 설정값이 코디 추천 로직에 반영

────────────────────────────────────────
13. 다크모드
────────────────────────────────────────
  - 상단 🌙 버튼으로 즉시 전환
  - 전체 UI 다크모드 적용
  - 선택값 브라우저에 저장 (재방문 시 유지)

────────────────────────────────────────
14. 색상 팔레트 표시 (ColorPalette)
────────────────────────────────────────
  - 오늘 코디에 어울리는 색상 조합 시각화
  - 날씨/스타일/기온에 따라 자동 생성
  - 색상 이름 및 HEX 코드 표시

────────────────────────────────────────
15. 일별 코디 알림 (NotificationSettings)
────────────────────────────────────────
  - 원하는 시간 설정 시 매일 푸시 알림
  - 브라우저 Notification API 사용
  - 알림 허용 / 해제 기능

────────────────────────────────────────
16. 온보딩 튜토리얼 (OnboardingTutorial)
────────────────────────────────────────
  - 최초 방문 시 4단계 사용법 안내
  - 건너뛰기 / 단계 이동 가능
  - 완료 후 다시 보기: 하단 "사용 가이드" 링크

────────────────────────────────────────
17. 도시 선택
────────────────────────────────────────
  - 서울·부산·대구·인천·광주·대전·수원 등 주요 도시 지원
  - 도시 변경 시 해당 지역 실시간 날씨로 전환

────────────────────────────────────────
18. 데이터 초기화
────────────────────────────────────────
  - "초기화" 버튼으로 프로필·옷장·기록 전체 삭제
  - 프로필 수정 기능 (프로필 버튼)


========================================
  기술 구현 포인트
========================================

  · 서버 없음    — 모든 데이터를 브라우저 LocalStorage에 저장
  · 무료 운영    — Open-Meteo(날씨), Groq(AI) 모두 무료 API 사용
  · Canvas API  — 서버 없이 브라우저에서 PNG 이미지 직접 생성
  · Web Share   — 모바일에서 네이티브 공유 시트 연동
  · PWA 준비    — 오프라인 대응 구조 (추후 확장 가능)
  · 반응형 UI   — 모바일/태블릿/데스크탑 모두 대응
